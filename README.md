# 📊 Power BI Data Analysis Project

## 📌 Overview
This project was completed as part of a **Data Technician Bootcamp**, focusing on using **Microsoft Power BI** to transform, analyze, and visualize data through interactive reports and dashboards.

The project demonstrates how raw datasets can be cleaned, modeled, and transformed into **meaningful insights** using Power BI tools such as **Power Query, DAX, and interactive visualisations**. The final report presents key insights from **retail and sales-style datasets**, helping stakeholders understand trends, performance, and patterns.

---

## 🎯 Project Objective

The objective of this project was to develop practical skills in:

- Data transformation
- Data modelling
- Dashboard design
- Interactive data exploration
- Communicating insights effectively

These skills are essential for **data analysts who need to turn complex datasets into meaningful business insights**.

---

## 🛠️ Skills Demonstrated

- 🔄 **Data Transformation & Cleaning with Power Query**
- 📊 **Interactive Report Development**
- 🧮 **DAX Calculated Columns and Measures**
- 🎛️ **Filters and Slicers for Data Exploration**
- 📈 **Trend Analysis and KPI Monitoring**
- 📖 **Data Storytelling through dashboards**
- 🧠 **Data Modelling and Relationships**

---

## 🔄 Data Preparation (Power Query)

Before visualisation, the dataset was cleaned and transformed using **Power Query**, including:

- Removing duplicates and unnecessary columns
- Handling missing or inconsistent data
- Renaming and formatting fields
- Creating structured tables ready for analysis

This ensures the dataset is **accurate, consistent, and ready for reporting**.

---

## 🧮 Data Modelling & DAX

Custom calculations were created using **DAX (Data Analysis Expressions)**:

- **Calculated Columns** to derive new attributes
- **Measures** to calculate aggregated metrics
- Performance indicators such as totals, averages, and percentages

These calculations allow deeper insights and more flexible reporting.

---

## 📉 Visualisations Used

A range of visualisations were used to present the data clearly and effectively:

| Visualisation | Purpose |
|---------------|--------|
| 📊 **Bar Charts** | Compare product or category performance |
| 📈 **Line Charts** | Show trends over time |
| 🥧 **Pie Charts** | Display distribution of sales categories |
| 🗺️ **Maps** | Visualise geographic sales patterns |
| 📋 **Tables & Cards** | Present key metrics and KPIs |

---

## 🎛️ Interactive Dashboard Features

The Power BI reports include interactive features such as:

- **Slicers** to filter data by category, region, or time
- **Cross-filtering between visualisations**
- **Dynamic updates across the entire report**
- **User-friendly layout for exploration**

This allows users to **interact with the data and explore insights dynamically**.

---

## 📖 Data Storytelling

A key focus of the project is **data storytelling** — presenting data in a way that clearly communicates insights.

The dashboard helps answer important questions such as:

- Which products or regions generate the most sales?
- How have sales trends changed over time?
- What patterns or relationships exist in the data?

By combining multiple visualisations into a single report, the project transforms raw data into **clear, actionable insights**.

---

## 🧰 Tools Used

- 📊 **Microsoft Power BI**
- 🔄 **Power Query (ETL)**
- 🧮 **DAX (Data Analysis Expressions)**
- 📄 **Excel / CSV datasets**
